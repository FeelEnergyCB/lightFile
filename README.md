# lightFile

### Customization of default input [type="file"]

### Demo:
https://mikitalisavets.me/light-file

### Usage:
```
lightFile({
  selector: '.file',
  customClass: 'lightFile',
  buttonText: 'Выбрать файл',
  deleteText: 'Удалить',
});
```

### Browser support:
IE 9+ / Firefox / Chrome  / Safari
